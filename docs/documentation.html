Introduction

What is OmniConnect Hub?

OmniConnect Hub is a premium React + TypeScript admin dashboard template designed for logistics, shipping, and supply-chain platforms. It ships with a stunning interactive 3D globe, live-updating KPI cards, advanced analytics charts, a carbon-footprint calculator, and a fully customizable multi-theme system — all powered by production-grade tooling.

Real‑Time Data

How the live data simulation works

The hook useRealTimeData drives the live KPI updates on the Dashboard. Every 2 seconds, it randomly selects 2–3 metrics and applies a tiny realistic delta — no full re-renders, no flicker.

MetricUpdate Logic
totalShipments+1 to +3 per tick (always increasing)
activeShipments±1 or ±2 fluctuation around baseline
delayedShipments±1, clamped between 5 and 28
fuelConsumption+2 to +5 per tick
co2Emission+1 to +3 per tick
systemEfficiency±0.1%, clamped between 92% and 99.9%

Run Analysis

Clicking Run Analysis pauses the ticker for 1.8 seconds (simulating computation), then improves systemEfficiency, co2Emission, and fuelConsumption, and returns a realistic route-optimization recommendation.

Adding a Custom Theme

Extend the 6 built-in palettes with your own brand colors

Open src/lib/themes.ts and append a new object to the BRAND_THEMES array:

// src/lib/themes.ts — add to BRAND_THEMES array
{
  id:           "myBrand",
  name:         "My Brand",
  description:  "Custom palette for XYZ Corp",
  primary:      "#ff6b35",
  secondary:    "#e63946",
  accent:       "#ff8c69",
  glow:         "rgba(255,107,53,0.5)",
  primaryRgb:   "255, 107, 53",
  secondaryRgb: "230, 57, 70",
}

That's it. The ThemeContext will automatically pick up the new theme and update both the CSS variables and the 3D globe materials — no other changes needed.

All demo data lives in src/lib/mockData.ts.

To connect real data, replace generateShipments() with API calls and update the useRealTimeData hook to fetch from your backend.

Routing

Client-side routing with Wouter

The project uses Wouter — a lightweight React router (~1.5 KB). All routes are declared in src/App.tsx inside a <Switch> block, wrapped in Framer Motion's <AnimatePresence> for smooth page transitions.

To add a new page: create a component in src/pages/, import it in App.tsx, add a <Route> entry, then add the link to Sidebar.tsx.

ℹ️ Base path: Configurable via the BASE_PATH environment variable — useful for sub-directory deployments (e.g. /demo/).

FAQ

Common questions

The 3D globe is slow — how do I improve performance?

The Globe3D component uses the useDevicePerformance hook to automatically lower route count, disable satellites, and reduce geometry detail on low-end devices. You can tune these thresholds inside the hook.

Can I replace the mock data with a real API?

Yes. Replace generateShipments() calls with @tanstack/react-query fetch calls. React Query is already set up in App.tsx — add your query hooks under src/hooks/.

How do I change the default theme?

Open src/lib/themes.ts and change DEFAULT_THEME_ID to any theme id (e.g. "ecoGreen").

The globe shows a blank canvas — what's wrong?

Make sure WebGL is enabled in your browser. On headless or CI environments WebGL is typically unavailable — this is expected behavior.

Can I use npm instead of pnpm?

Yes. Remove the "preinstall" script from the root package.json, then run npm install from the root.
